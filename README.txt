README for AI_Wargame: 

These are the steps to run AI_Warzone:

1) open a command prompt and cd to the directory of the application
2) To play the game, you will need to type a command similar to the following: python ai_wargame.py --game_type manual --max_turn 100
2.1) where:
	 --game_type is the type of play mode
	 --max_turn is the max amount of turns the attacker has to try to beat the defender
